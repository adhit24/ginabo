# 📄 design.md — Ginabo Website

## 1. Overview

**Project Name:** Ginabo Website
**Type:** Conversion-focused brand website (DTC skincare)
**Goal:**

* Drive First Try purchase
* Build trust
* Increase repeat order

**Core Positioning:**

> Nutrition Skin for Active Woman

---

## 2. Design Principles

### 2.1 Brand Feel

* Calm
* Clean
* Premium but approachable
* Not overly feminine
* Not flashy / not “promo marketplace”

### 2.2 Visual Direction

* Soft lighting
* Real lifestyle (active women)
* Minimal layout
* Generous whitespace

---

## 3. Color System

```css
Primary:        #5B4B8A
Secondary:      #C6B7E2
Accent:         #7A6AD8
Background:     #F8F7FB
Text Primary:   #2E2A3B
White:          #FFFFFF
```

### Usage Rules:

* Primary → buttons, highlights
* Background → full page
* Secondary → section background
* Accent → hover / small highlights

---

## 4. Typography

### Font Family

* Primary: Montserrat

### Hierarchy

```css
H1: 40–48px / bold
H2: 28–32px / semi-bold
H3: 20–24px / medium
Body: 14–16px / regular
Caption: 12px
```

### Rules:

* Line height: 140–160%
* Avoid long paragraphs
* Max 2–3 lines per block

---

## 5. Layout System

### Container

```css
max-width: 1200px
padding: 0 24px
margin: auto
```

### Grid

* Desktop: 12 columns
* Tablet: 6 columns
* Mobile: 1 column

### Spacing

```css
section spacing: 80–120px
element spacing: 16–32px
border radius: 10px
```
